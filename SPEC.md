# 流氓广告 - 产品设计规格书

## 1. 项目概述

### 项目名称与定位
- **应用名称**：流氓广告
- **项目类型**：Android讽刺娱乐应用
- **核心定位**：通过夸张、幽默的方式讽刺国内APP的各种广告形式，让用户在"被广告支配"的体验中获得乐趣
- **目标用户**：被各种广告骚扰的普通手机用户

### 核心价值主张
- "让你体验被广告支配的恐惧！"
- "原来广告可以这么烦人！"
- "看完这个APP，我觉得其他APP的广告也没那么讨厌了"

### 用户体验目标
- 沉浸式体验各种烦人广告的"真实"效果
- 通过游戏化机制获得成就感和乐趣
- 在幽默讽刺中释放对广告的不满情绪

## 2. UI/UX设计规格

### 整体视觉风格
- **设计语言**：夸张讽刺风格，融合真实广告的视觉元素
- **主色调**：
  - 主色：#FF5722（警示橙）- 代表广告的"危险"和"烦人"
  - 辅助色：#4CAF50（伪装绿）- 模拟"安全"、"关闭"按钮的欺骗性
  - 强调色：#2196F3（点击蓝）- 诱导用户点击
  - 背景色：#FFFFFF（白底）或 #1A1A1A（深色模式）
  - 警告色：#F44336（弹窗红）
- **字体**：
  - 主字体：Roboto
  - 数字/分数：Roboto Mono
  - 讽刺文案：思源黑体 Bold

### 配色方案详细说明
```
警示橙 #FF5722：主色调，代表广告的烦人程度
伪装绿 #4CAF50：模拟真实广告中"安全"、"跳过"按钮
诱导蓝 #2196F3：模拟诱导点击的按钮
欺骗灰 #9E9E9E：模拟"稍后再说"、"取消"等次要选项
弹窗红 #F44336：弹窗和警告元素
```

### 布局结构
```
应用架构：
├── 首页 - 广告博物馆
│   ├── 广告类型分类标签（强干扰/中干扰/弱干扰/系统级/特殊交互/其他）
│   ├── 广告卡片列表
│   └── 顶部状态栏（广告忍耐度 + 广告币）
├── 广告体验页面
│   ├── 模拟广告展示区
│   ├── 讽刺文案区
│   └── 关闭/跳过按钮（如果存在）
├── 成就中心
│   ├── 已解锁成就列表
│   └── 成就进度
└── 关于/设置
    ├── 深色模式切换
    ├── 音效开关
    └── 关于开发者（搞笑文案）
```

### 界面交互规格

#### 首页 - 广告博物馆
- **布局**：可滚动的卡片列表，按广告类型分类
- **广告卡片**：
  - 左侧：图标（emoji或简笔画图标）
  - 中间：广告名称 + 一行讽刺性描述
  - 右侧：体验次数统计
  - 点击效果：卡片轻微缩放 + 阴影加深
- **顶部状态栏**：
  - 广告忍耐度进度条（从100%开始，随体验广告下降）
  - 广告币数量（用于解锁特殊成就）
- **下拉刷新**：显示"正在加载广告..."的讽刺文案

#### 广告体验页面
- **布局**：全屏模拟真实广告场景
- **交互元素**：
  - 跳过按钮（仅在倒计时结束后显示）
  - 关闭按钮（极小且位置刁钻）
  - 诱导点击区域（模拟真实广告的陷阱）
- **体验流程**：
  1. 进入页面 → 显示广告
  2. 用户可尝试关闭/跳过
  3. 广告展示完成后 → 显示讽刺文案
  4. 返回首页 → 更新统计数据

#### 成就系统
- **成就类型**：
  - "广告幸存者"：成功关闭5个弹窗
  - "摇一摇绝缘体"：在摇一摇广告中存活30秒
  - "忍耐之王"：广告忍耐度归零前体验完所有广告
  - "眼疾手快"：在3秒内点击跳过按钮
  - "被坑专家"：误触诱导区域10次
- **奖励机制**：
  - 每个成就解锁可获得广告币
  - 全部成就解锁可获得"广告大师"称号

### 动画效果规格
- **弹窗出现**：从中心缩放 + 轻微抖动
- **跳过按钮**：3秒倒计时圆圈动画
- **诱导点击**：脉冲闪烁效果
- **成就解锁**：金色粒子爆炸效果
- **页面切换**：共享元素过渡动画
- **广告忍耐度**：渐变色进度条动画

### 音效规格（可选）
- 弹窗弹出："叮咚"提示音
- 诱导点击：诱惑的"咔哒"声
- 成功关闭：解压的"呼"声
- 成就解锁：胜利号角

## 3. 功能规格

### 核心功能列表

#### 3.1 广告类型展示
- 展示32种广告形式的完整列表
- 支持分类筛选：
  - 强干扰型（5种）：开屏、插屏、全屏视频、悬浮窗、弹窗
  - 中干扰型（6种）：横幅、信息流、原生、搜索结果、推荐位、贴片
  - 弱干扰型（6种）：激励视频、积分墙、试玩、内容植入、评论区、个人中心
  - 系统级/跨应用（5种）：通知栏、锁屏、桌面图标、应用内更新、系统功能伪装
  - 特殊交互型（6种）：摇一摇、滑动解锁、语音、AR、弹幕、表情包
  - 其他（4种）：支付成功页、分享页、退弹、空状态

#### 3.2 广告体验功能
每个广告类型都有独立的体验页面，包含：
- 高度还原的广告展示效果
- 可交互的关闭/跳过机制
- 倒计时或等待机制
- 体验完成后的讽刺文案

#### 3.3 游戏化系统
- **广告忍耐度**：
  - 初始值：100%
  - 每体验一种广告：根据烦人程度扣除
  - 恢复方式：观看激励视频广告（讽刺）
- **广告币**：
  - 初始值：0
  - 获取方式：体验广告、解锁成就
  - 用途：解锁特殊成就、购买"跳过所有广告"功能（极度讽刺）
- **成就系统**：
  - 10个可解锁成就
  - 每个成就附带搞笑文案

#### 3.4 数据持久化
- 本地存储用户数据（SharedPreferences）
- 存储内容：
  - 体验过的广告列表
  - 广告币数量
  - 已解锁成就
  - 广告忍耐度
  - 设置偏好

### 32种广告形式详细清单

#### 强干扰型广告

**1. 开屏广告**
- 体验效果：3-5秒全屏广告，倒计时结束后才显示跳过按钮
- 讽刺文案："你以为倒计时结束就能跳过了？太天真了！"
- 交互：摇一摇手机会触发跳转（全屏震动效果模拟）

**2. 插屏广告**
- 体验效果：随机时机弹出全屏广告
- 讽刺文案："正看得开心？不好意思，广告来了！"
- 交互：点击屏幕任意位置都可能触发

**3. 全屏视频广告**
- 体验效果：15秒视频广告，5秒后可跳过
- 讽刺文案："感谢您观看广告，请继续等待..."
- 交互：倒计时结束后点击跳过按钮

**4. 悬浮窗广告**
- 体验效果：半透明悬浮球，可拖动但关闭按钮极小
- 讽刺文案："你永远关不掉我~"
- 交互：拖动悬浮球，尝试点击极小的关闭按钮

**5. 弹窗广告**
- 体验效果：突然弹出的对话框，"关闭"按钮设计成领取样式
- 讽刺文案："恭喜你获得了...一个关闭按钮！"
- 交互：识别哪个才是真正的关闭按钮

#### 中干扰型广告

**6. 横幅广告**
- 体验效果：顶部或底部固定显示的广告条
- 讽刺文案："我不影响你看内容，真的。"
- 交互：尝试关闭横幅

**7. 信息流广告**
- 体验效果：伪装成普通内容的广告卡片
- 讽刺文案："这条不是广告，你信吗？"
- 交互：识别哪些是广告哪些是真实内容

**8. 原生广告**
- 体验效果：与APP风格完全融合的广告
- 讽刺文案："我是广告，但我假装不是~"
- 交互：寻找微小的"广告"标识

**9. 搜索结果广告**
- 体验效果：搜索结果顶部带有"广告"标签的内容
- 讽刺文案："你搜的是这个，我推的是那个~"
- 交互：区分搜索结果和广告

**10. 推荐位广告**
- 体验效果：猜你喜欢、为你推荐板块中的广告
- 讽刺文案："我猜你喜欢这个广告！"
- 交互：发现推荐列表中的广告

**11. 贴片广告**
- 体验效果：视频前、中、后插入的广告
- 讽刺文案："视频开始前，先看60秒广告吧~"
- 交互：等待广告播放完成或点击跳过

#### 弱干扰但隐蔽型广告

**12. 激励视频广告**
- 体验效果：主动观看可获得奖励
- 讽刺文案："看广告还能赚钱？赚的是寂寞..."
- 交互：选择观看或不观看广告

**13. 积分墙广告**
- 体验效果：下载APP、注册账号换取积分
- 讽刺文案："想要奖励？先把你的隐私交出来！"
- 交互：选择任务并完成（模拟）

**14. 试玩广告**
- 体验效果：可直接体验的游戏广告
- 讽刺文案："试玩30秒后，你更想玩了！但要付费~"
- 交互：体验游戏片段

**15. 内容植入广告**
- 体验效果：软文、种草笔记形式的广告
- 讽刺文案："这篇测评真的很客观...才怪！"
- 交互：识别植入内容

**16. 评论区广告**
- 体验效果：伪装成用户评论的广告
- 讽刺文案："这条评论是假的，但你信了~"
- 交互：识别评论区的广告

**17. 个人中心广告**
- 体验效果：设置页面中的广告
- 讽刺文案："你都在我的个人中心了，看个广告呗~"
- 交互：发现并关闭个人中心广告

#### 系统级/跨应用广告

**18. 通知栏广告**
- 体验效果：模拟系统通知推送的广告
- 讽刺文案："就算你没打开APP，我也能骚扰你！"
- 交互：模拟点击通知

**19. 锁屏广告**

- 讽刺文案："解锁前，先看个广告？"
- 交互：模拟解锁手机

**20. 桌面图标广告**
- 体验效果：桌面图标上的红点或角标
- 讽刺文案："小红点强迫症的噩梦~"
- 交互：尝试消除角标

**21. 应用内更新广告**
- 体验效果：伪装成更新提示的广告
- 讽刺文案："点更新？恭喜你，你下载了另一个APP！"
- 交互：识别真假更新提示

**22. 系统功能伪装广告**
- 体验效果：伪装成WiFi、电池等系统提示
- 讽刺文案："你的手机很危险！...但其实没有~"
- 交互：识别伪装提示

#### 特殊交互型广告

**23. 摇一摇广告**
- 体验效果：晃动手机触发跳转
- 讽刺文案："你摇什么摇？摇也跳不过去~"
- 交互：尝试不摇手机或快速摇动

**24. 滑动解锁广告**
- 体验效果：滑动特定区域才能关闭
- 讽刺文案："解锁广告的解锁方式~"
- 交互：完成滑动解锁

- 体验效果：自动播放的语音广告
- 讽刺文案："想安静？不存在的！"
- 交互：尝试静音或关闭

**26. AR广告**
- 体验效果：AR滤镜中的广告
- 讽刺文案："AR让你无处可逃~"
- 交互：与AR广告互动

**27. 弹幕广告**
- 体验效果：视频弹幕中的广告
- 讽刺文案："弹幕广告，躲都躲不掉~"
- 交互：等待广告弹幕飘过

**28. 表情包广告**
- 体验效果：表情包形式的广告
- 讽刺文案："可爱吗？可爱就对了！"
- 交互：发送表情包广告

#### 其他广告

**29. 支付成功页广告**
- 体验效果：完成支付后展示的广告
- 讽刺文案："你都付款了，再看一眼广告呗~"
- 交互：跳过或查看广告

**30. 分享页广告**
- 体验效果：分享内容时展示的广告
- 讽刺文案："分享传播，广告先行~"
- 交互：模拟分享流程

**31. 退弹广告**
- 体验效果：退出时弹出的广告
- 讽刺文案："别走！再看一个广告！"
- 交互：成功退出或被诱导

**32. 空状态广告**
- 体验效果：搜索结果为空时展示的广告
- 讽刺文案："没找到内容？那就看广告吧！"
- 交互：体验空状态场景
